# Lumi 🤖
A conversational AI built in Python.

## Setup
```bash
pip install -r requirements.txt
cp .env .env.local  # add your API keys
python main.py
```

## Structure
- `src/`      — core logic (chat, memory, prompts, tools, personality)
- `data/`     — conversations, knowledge base, personas
- `models/`   — weights and adapters
- `configs/`  — yaml config files
- `tests/`    — unit and integration tests
